## Timing Guidance

- **Rising pattern + your domain** → timely, submit to top venue
- **Stable high** → competitive, need deeper insight
- **Falling** → saturated unless qualitative breakthrough
- **Never appeared in your domain** → high risk / high reward
- **Community-driven (low PC bias)** → focus on practical utility, infrastructure value
- **Strong PC preference** → focus on theoretical rigor, formal guarantees
